# 🚐 Camper Rental App

A modern web application for browsing and booking camper vans.

## 📦 Features

- Browse campers catalog
- Filter by:
  - location
  - vehicle type
  - engine
  - transmission

- View detailed camper information
- Image gallery with slider
- Reviews with rating system ⭐
- Booking form with validation
- Success / error popup after booking

## 🛠 Tech Stack

- Next.js (App Router)
- TypeScript
- React Hook Form
- CSS Modules
- Swiper.js
- React Icons

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## 📁 Project Structure

- `/app` – routing (Next.js App Router)
- `/components` – reusable UI components
- `/lib` – API & validation logic
- `/types` – TypeScript types
- `/public` – images & assets
